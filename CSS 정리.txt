-selector
*: html의 모든 요소들
img: 모든 이미지
button: 모든 버튼
,(h1,h2): h1과 h2



-ID selector(선택자): css의 모든 요소에 훅을 제공: #
#logout(id){
	color: orange;
	height: 200px;
}
여러개 중 특정한 것만 바꾸고 싶을 때 <button id=" ">로 id 지정해주고 css에서 #(id) {}로 씀. 
***id는 중복되면 안됨( 변수 같은 개념 ), id는 최소화 하는 게 좋다.


CLASS selector(id와 비슷한 개념인데 더 다양한 곳에 쓰임 ): .
.tag(class){
	color: red;
}


-descendant(자손) selector: 띄어쓰기
li a{ 					//li 태그에 중첩된 모든 자손 요소를 선택
	color: blue;
}
li a는 li 속 앵커 태그이고 li,a는 li와 앵커 태그 이다. 따라서 li a 는 <li>태그는 선택X
***section에 클래스를 부여하고 각각에 


-ADJACENT selector(인접 선택자): +
h2 + button    => 모든 <h2> 태그 바로 다음에 오는 <button>


-DIRECT CHILD selector(직계 자손 선택자): >
div > li       => <div> 속 직계 자손인 <li> 태그만 선택 (직계 = 바로 하위 단계)


-ATTRIBUTE selector(속성 선택자): []
...<type> 속성에서 본 <input> 요소
input {} : html 속의 모든 input 요소들 바꿈
input[type="password"]{} : html의 password 속의 input 요소 바꿈

section[class="post"] == section.post

특정 앵커 태그만 선택하고 싶으면 a[href*="google"]{} : href 속성에 google 이 포함된 모든 앵커 태그만 선택

앵커태그 & 속성 선택자
***a[href=""] : ""와 동일한 앵커태그   =>  이게 많이 쓰임
a[href*=""] :""가 포함된 모든 href
a[href$=""] :""로 끝나는 href
a[class~=""]



가상 클래스(pseudo class): 선택자 끝에 붙여 상태를 특정.     콜론을 사용함.

a:hover{}
button:hover{}
.post button:hover{}   :  클래스 포스트 안에 있는 버튼 특정


active : 무언가가 활성화 되어있다는 표시(눌렀을 때 다른 색 표시)

checked

nth-of-type(정수 or 2n,3n...): 특정 숫자의 선택자 마다 상태 특정.
